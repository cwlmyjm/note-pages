# note-pages

## 主题定义

>在views文件夹里面存在主题文件夹themes，比如说默认主题default，里面包含该主题的前端文件，除此之外还有一个配置文件config.json，该文件记录了不同页面需要加载的前段文件。
    
    default主题的目录结构
    
    default
    |-content.php
    |-body.php
    |-config.json
